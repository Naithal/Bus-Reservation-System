# Bus Reservation System (Java)

A simple console-based Bus Reservation System built using Java. It allows users to book bus tickets and admin to manage buses.

---

## Features

### User
- Register and login
- Search buses
- Book tickets
- Make payment (simulation)
- View ticket

### Admin
- Login (admin/admin123)
- Add, update, delete buses
- View all buses
- View bookings

---
